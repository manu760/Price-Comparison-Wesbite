# Price-Comparison-Wesbite
Price comparison website using web scraping with selenium.
Technology used -   Maven, Spring, Hibernate to implement back-end.
Database -          MySQL
Restful API-        Node.js
Front-end-          HTML, CSS and JavaScript
List of Scraped Stores-
1.Ebay
2.TechInTheBasket
3.JDWilliams
4.Ebuyer
5.Currys
